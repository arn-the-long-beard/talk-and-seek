* Summary of set up

Download

    git clone git clone https://github.com/arn-the-long-beard/talk-and-seek.git

Go to the directory
   
    cd talk-and-seek
    
Install Npm Packages
    
    npm install 
    
Start the server
It is working on http because of the wikiJs. Usually I run everything on https.
    
    npm start
